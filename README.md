# \<kemet-drawer>

## Installation
```bash
npm i @kemet/kemet-drawer
```

## Usage
```js
import '@kemet/kemet-drawer/kemet-drawer.js';;
```

```html
<kemet-drawer effect="slide">
  <nav slot="navigation">
    [YOUR NAV HERE]
  </nav>
  <section slot="content">
    [YOUR CONTENT HERE]
  </section>
</kemet-drawer>
```

[Click here](http://kemet.online/drawer) for more docs.
